ABOUT THIS WEBSITE

This website was created in October 2018 when I was an exchange student in Offenburg, Germany. I was taking a course on web applications and I made this website as a way to share the applications that I was developing with my friends and family. Soon, the whole website became a project itself, and I started improving it and adapting it to my likes and needs. Now the website is mostly used as a place where I can test new web applications and make school projects. It also serves as a portfolio and a place where I can display the things I have been working on.

When it was first started, this website was hosted on an AWS EC2 instance using a LAMP stack and everything was (and probably still is) really crude. I was starting to learn HTML, CSS and PHP and this was intended as a learning exercise for improving my skills.

Most of the applications here were build from scratch, without the use of any tool other than programming libraries and a text editor. This may change over time, as I plan to test new technologies and frameworks to improve this site. Therefore, do note that this website is a work in progress.
